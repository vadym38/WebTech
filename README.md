# Tutors Finder SPA

Навчальний SPA-додаток на ReactJS для теми "Сервіс пошуку репетиторів".

## Локальний запуск

1. У XAMPP увімкнути MySQL.
2. Переконатися, що база даних називається `tutors`.
3. Запустити файл `start_spa.bat`.
4. Відкрити адресу `http://localhost:8080/index.html`.

## Хостинг на Render

Проєкт підготовлено до деплою на Render через Docker.

1. Завантажити папку `spa_lab` у GitHub-репозиторій.
2. У Render створити `New Web Service`.
3. Підключити GitHub-репозиторій.
4. Обрати Docker environment.
5. Після деплою відкрити адресу, яку видасть Render.

Якщо на хостингу не підключена MySQL-база, API автоматично використовує демо-дані з JSON-сховища. Для підключення MySQL можна задати змінні середовища `DB_HOST`, `DB_USER`, `DB_PASS`, `DB_NAME`.

## Що реалізовано

- SPA на ReactJS без перезавантаження сторінок.
- Навігація між сторінками: головна, уроки, редагування, авторизація.
- Навчальна авторизація зі збереженням користувача в `localStorage`.
- Компонент `LocalCounter` з власним станом і подіями.
- Завантаження даних з API у форматі JSON через AJAX `fetch`.
- Відображення уроків із підстановкою даних зі зв'язаних таблиць.
- Фільтрування і сортування уроків.
- Форма редагування уроку через `PUT`-запит.
- Управління станом додатку через `useReducer`.
